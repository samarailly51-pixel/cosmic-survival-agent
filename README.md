# 未来星际生存指南 AI Agent

一个零依赖的静态 Web 原型：把未来太空探索、生存决策、资源状态和 AI Agent 对话做成一张可交互的星云关系图。

当前版本已经不局限于火星，而是以“深空星图”为核心界面：中心 AI 导航核、发散光丝、漂浮节点、星区标签和底部星图控制条共同构成一个宇宙探索式交互体验。

## 在线发布

这个项目可以直接部署到 GitHub Pages。仓库推送到 GitHub 后，在仓库设置里启用 Pages，或使用本项目内置的 GitHub Actions 工作流自动发布。

## 本地预览

推荐使用本地静态服务器预览：

```powershell
py -m http.server 8765 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:8765/index.html
```

也可以直接用浏览器打开 `index.html`，但本地服务器更接近发布后的环境。

## 功能

- 星云式关系图：中心 AI 核心、漂浮星点、中文星区标签和弯曲光丝
- 可交互节点：点击星图节点触发 AI 分析、对话和资源变化
- 星图控制条：调节粒子大小、线条亮度、标签显示范围、动画暂停/继续
- 生存资源面板：氧气、水循环、电力、心理韧性、辐射风险、粒子风暴
- 事件决策卡：不同选择会影响资源数值并推进事件
- 本地状态保存：浏览器会通过 `localStorage` 保留当前状态，右上角 `R` 可重置

```text
├── .nojekyll       # GitHub Pages 静态发布标记
└── .github/
    └── workflows/
        └── pages.yml
```
